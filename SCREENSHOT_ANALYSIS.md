# 截圖分析結果

## 後台 Dashboard 截圖（共 ~10 張）

### 1. 主儀表板 (Dashboard)
- 頂部統計卡片：今日預約數、本月營收、活躍會員數、待處理訂單
- 預約趨勢折線圖（7天/30天）
- 最近預約列表（含狀態標籤：待確認/已確認/已完成）
- 側邊欄導航：儀表板、預約管理、客戶管理、訂單管理、商品管理、術後照護、時段設定、統計報表、系統設定

### 2. 預約管理 (Appointments)
- 日曆視圖 + 列表視圖切換
- 預約卡片：客戶名稱、療程、時間、狀態
- 操作按鈕：確認/取消/改期
- 篩選器：日期範圍、狀態、療程類型

### 3. 客戶管理 (Customers)
- 客戶列表表格：姓名、電話、LINE 綁定狀態、會員等級、消費總額
- 搜尋與篩選
- 客戶詳情頁：基本資料、預約記錄、消費記錄、標籤

### 4. 訂單管理 (Orders)
- 訂單列表：訂單編號、客戶、金額、狀態、日期
- 狀態篩選：待確認/已確認/已出貨/已完成/已取消

### 5. 時段設定 (Slot Management)
- 日曆格式顯示各時段狀態
- 可設定每時段人數上限
- 客滿自動標記

### 6. 統計報表
- 營收統計圖表
- 預約來源分析
- 會員註冊趨勢
- 時段使用率

## 前端 Landing Page 截圖（1-2 張）

- 服務介紹

## LINE LIFF 操作端截圖（5-6 張）
### 六宮格圖文選單
- 左上：VIP 預約系統
- 中上：VIP 商城
- 右上：術後護理知識包（圖卡推播）
- 左下：減脂追蹤
- 中下：VIP 會員中心
- 右下：客戶留言/聯絡我們

- 推播術後注意事項 Flex Message
